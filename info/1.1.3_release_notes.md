# Clide 1.1.3 Release Notes

### Release Date
- February 14, 2026

### Highlights
- Fixed Windows app startup failure after installation.
- Corrected installer launcher configuration for JavaFX startup.
- Updated persistent data path to user profile (`~/.clide`) to avoid write issues under `Program Files`.
- Improved installer build reliability and versioning alignment.
- Added MSI fallback installer support in addition to EXE.

### Installer Artifacts
- `Clide-1.1.3.exe`
- `Clide-1.1.3.msi`

### Documentation and Website
- Added/updated landing page in `docs/` with:
  - Demo GIF preview
  - About Me section
  - GitHub release download button
  - Clide favicon
  - Footer attribution: `by SACHIN SHANKAR`
- Moved installer guide to `info/INSTALLER.md`.
- Visit website at [tinyurl.com/clide-dev](https://www.tinyurl.com/clide-dev)

### Notes
- If a previous 1.1.0 install exists, uninstall it before installing 1.1.3 (if looking to change to .msi)
- If on Clide 1.1.0 and staying on .exe, then do not uninstall unless you really want to 
- Click `approve` to allow Clide to accessing your network through Windows or else the app will not load

### Troubleshooting
- If any issues arise, reinstall the app
- If that doesn't help, head to [our GitHub Discussions](https://tinyurl.com/discuss-clide)
